Machine Learing Model for Stock Price Predictions
This is an Machine learning Model for Stock price predictions which gets hold of data from the past sixty days of price movements to in order to predict next day's price. Data Fetched from Yahoo Finance API, Libraries used are numpy, scikit-learn, pandas, tensorflow, pandas_datareader. The model gives you a good enough prediction corresponding to the short term price movements. PS: This is only for programming learning perspective Advised to do not use it for predicting/investing in the stocks. Here's the Git Repository for the Model.
